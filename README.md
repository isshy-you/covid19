# Usage

normal execution
```PowerShell
git clone https://github.com/isshy-you/covid19
cd covid19
python covid19.py
```
compile
```PowerShell
cd covid19
pip install pyinstaller
pyinstaller covid19.py --name covid19 --onefile --clean
```
execution by compiled
```PowerShell
.\dist\covid19.exe
```
